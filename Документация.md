# 📦 Финальная документация проекта "FoodWise"

## 🚀 Запуск проекта

1. Установите зависимости:

```
pip install -r requirements.txt
```

2. Запустите приложение:

```
python app.py
```

3. Перейдите в браузер: `http://127.0.0.1:5000/`
